Customer Data Analysis with PySpark

This project demonstrates data cleaning, validation, and analysis using PySpark on a Hadoop cluster.

Features:

Read Excel sheets with a predefined schema

Convert date columns to proper DateType

Validate dates to identify incorrect formats

Standardize text columns to uppercase/lowercase

Create Spark SQL tables from DataFrames for querying

Aggregate and analyze active customers by state

Tech Stack:

PySpark

Hadoop (HDFS)

Jupyter Notebook

This repo showcases best practices for handling large datasets, data validation, and scalable analytics using Spark.
